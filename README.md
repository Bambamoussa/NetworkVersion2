# NetworkKM

## Version 2

### Les fonctionnalités

* Initialisation du graphe par défaut
* Fonctionnalités principales du menu (Ajout/Edition de noeuds/arcs)
* Sous-menu pour éditer un noeud ou l’étiquette d’un arc
* Arcs en ligne droite non orientés
* Utilisation de ressources :
* Pour les dimensions quand c’est utile
* Pour les strings quand c’est traduisible
* Envoi par email
* Reset du graphe

+++

* Orientation et courbure des arcs
* Modifier la courbure d’un arc au touch
* Etiquettes des arcs parfaitement dessinées (cf. sujet du TP)
* Prise en compte de la rotation (Le graphe ne doit pas être déformé et être entièrement accessible)
* Sauvegarde / Import de graphes
* Dessiner les flèches sur le bord des noeuds
* Implémentation du zoom
